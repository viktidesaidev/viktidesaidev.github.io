StalkTheStock : A Stock Price Predicting Web Application


With a team of 5, this web-application was created with the vision to help stock-traders predict short-term (one minute) and long term (one-day) stock priices. The front end was scripted in HTML, CSS and Javascript with the aid of Jquery libraries and AJAX. The backend which is the backbone of most web-applications was scripted in Python and PHP. The algorithms used for long-term and short-term predictions were Support-Vector Machine , Bayesian Predictor and Artificial Neural Networks. RestoLingo : A Restaurant Automation Web based Mobile Application


We built a web based mobile application for restaurants. Building a web-application was the course-requirement and we abided by it. Our motto was creating a user-friendly application for a conventional restaurant. With a vision to make the traditional pen-paper method of taking down orders obsolete, we included features such as order take-down. The application also includes features such as viewing available tables, viewing the bill, sending notifcations to the chef about the order, sending notificaions to the waiter if the dish is ready and many more. To make this application we used the Python - Django framework which made our work easy. We executed the project using Agile Methodology and had mock sprints to get the feel of working in a software industry. The application prototype was made mobile and tablet responsive only. Travellify: : A trip planning Android application


Android Applications are always fun to use. Coding them however, is always a challenge. We made Travellify with the purpose of creating a trip planner. The user can enter the source and destination of his/her trip. The shortest route is shown and on the path the most popular destinations are marked. On clicking a location pin the pictures clicked at that locateion are displayed. In the developement phase we tested the application inside our college campus and it worked. The application also had features like search by tag, add friends, tag suggestions. We used android studio to code the application. Google Maps API was used for integrating Maps in the application, clarifai API was used for image recognition and tag suggestions.

Implementation of Sequence Alignment Algorithms


To study data structures and alogorithms in depth, I studied sequence alignment algorithms like Needleman-Wunsch, Smith-Waterman, and Hirschberg's algorithms in order to. These algorithms are used widely in DNA research to study the similarities among various living beings. Hence the dataset we used was a collection of genetic DNA sequences. Implementing these algorithms gave me a sense of importance of data structures in the world of programming. It also gave me an insight about how important algorithms can be in terms of time and space complexities to achieve optimum performance and quick results. The algorithms were coded in Python. After the implementation I also carried out analysis with respect to the Big-O Notation and Doubling Hypothesis.
